# Sunflower Protocol: Ephemeral State Routing (ESR)

**Hardened, non-static routing for resilient decentralized infrastructure.**

## Overview

The Sunflower Protocol implements **Ephemeral State Routing (ESR)** — a dynamic routing system that eliminates permanent attack surfaces. Traditional static routing creates predictable vulnerabilities. ESR makes routes time-bound, sharded, and adaptive for sovereign resilience in energy grids, DePIN networks, and high-stakes infrastructure.

Built solo for the **next billion users** in a decentralized world.

## Core Features

- **Node Persistence**: Time-bound sessions that expire automatically.
- **State Sharding**: Distributed fragmentation for privacy and fault tolerance.
- **Traffic Pathing**: Dynamic heuristic routing prioritizing uptime and trust.
- **OTA Injection**: Seamless protocol upgrades with zero downtime.
